# SmartMerca 🛒

Sistema de gestión de inventario para supermercado.

## Tecnologías
- **Frontend:** React
- **Backend:** Python + Flask
- **Base de datos:** MySQL

## Instalación

### Backend
```bash
cd backend
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

### Frontend
```bash
cd frontend-react
npm install
npm start
```

## Roles
- **Administrador:** Gestión completa del sistema
- **Cajero:** Consulta de productos en inventario
